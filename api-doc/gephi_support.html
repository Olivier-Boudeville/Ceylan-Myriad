Support for the Gephi facilities.

Description

Support for the Gephi facilities.

Gephi is an open source tool to visualise and explore all kinds of graphs and networks.

See gephi_support_test.erl for the corresponding test, and the Gephi official website https://gephi.org/ for further information.

For the vast majority of the services defined here, an instance of a Gephi server is expected to have been configured and launched beforehand, both according to our conventions.
